1. まえがき
    DeskList をダウンロードして戴き有り難うございます。
    デスクトップが SysListView32 であることを利用して色々なスタイル
    に変更します。[大きいアイコン][小さいアイコン][一覧][詳細]はもちろん
    グリッド表示や単一行選択も可能になります。
    同梱のＤＬＬを利用するとグリッドに吸着したりシェルの動作を変更したり
    と様々なデスクトップカスタマイズが可能になります。

2. インストール及び使用方法
    解凍した次の3つのファイルを同一フォルダに入れて、DeskList.exe を実行してください。
    (1) DeskList.exe    本プログラムの実行ファイル
    (2) DeskHook.dll    フックＤＬＬ
    (3) Readme.txt      このファイル

3. 使い方
    起動するとタスクトレイにアイコンが表示されます。あとはアイコンを右クリック
    してメニュー選択してデスクトップスタイルを変更します。
    また左クリックするとデスクトップアイコンのランチメニューが表示されます。

[操作方法]
操作は全てマウスのみで行います。
タスクトレイを右クリックするとコントロールメニューが
表示され、左クリックするとランチメニューが表示されます。
以下にその動作を説明します。

★コントロールメニュー操作方法

  大きいアイコン ... 画面のアイコンを大きいアイコンで表示
  小さいアイコン ... 画面のアイコンを小さいアイコンで表示
  一覧 ... 画面のアイコンを一覧で表示
  詳細 ... 画面のアイコンを詳細で表示 [W2K未対応]
  ----------------
  [標準]
    ヘッダ非表示     ... 詳細表示時にヘッダ表示をしません
    スクロール非表示 ... 詳細表示時にスクロール表示をしません
  [拡張]
    グリッドの表示             ... 詳細表示時にグリッドを表示します
    サブアイテムイメージの表示 ... 詳細表示時にサブアイテムアイコンを
                                   表示します(通常のアイコンはもってません)
    チェックボックスの表示     ... 詳細表示時にチェックボックスを表示します
                                   (シェルの操作上で使い道はありません)
    ヘッダ項目の入替           ... 詳細表示時にヘッダを入れ替えることができます
    単一行の全選択             ... 詳細表示時に単一行として選択することができます
    ----------------------------
    マウス位置による選択   ... マウス位置によってアイコンを選択します
    シングルクリックで開く ... シングルクリックでアイコンを開きます
    ダブルクリックで開く   ... ダブルクリックでアイコンを開きます
    ----------------------------
    フラットスクロールバー ... スクロールバーを非立体的に表示します
    リージョナル           ... 未対応
    インフォチップ         ... アイコンにツールチップで情報を添付します
    アンダーラインホット   ... アクティブなアイテムに下線を表示します
    アンダーラインコールド ... 非アクティブなアイテムに下線を表示します
    マルチワークエリア     ... 未対応
  ----------------
  [フック]
    デスクトップをグリッドに吸着させる                 ... アイコンのずれを修正します
    フォルダの余白をダブルクリックすると一階層上がる   ... シェルの空白部をダブルクリック
                                                           すると一階層上がります
    画面をダブルクリックするとエクスプローラを起動する ... 画面の空白部をダブルクリックすると
                                                           エクスプローラを起動します
  ----------------
  [アイコンレイアウト]
    読込         ... 以前に保存したアイコン位置を読み込みます
    保存         ... 現在のアイコン位置を保存します
    起動時に読込 ... 起動時に保存しておいたアイコン位置を読み込みます
  ----------------
  [スタイル]
    読込         ... 以前に保存したスタイルを読み込みます
    保存         ... 現在のスタイルを保存します
    起動時に読込 ... 起動時に保存しておいたスタイルを読み込みます
  ----------------
  終了 ... DeskList を終了します

★ランチメニュー操作方法

  ウィンドウを最小化 ... 現在表示中の全てのウィンドウを非表示にします
  ----------------
  <デスクトップのアイコン> ... 選択したファイルを関連起動します
